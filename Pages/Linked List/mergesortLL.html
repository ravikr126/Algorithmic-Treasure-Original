Merge Sort for Linked List

Last updated: 29th Aug, 2020

        

Problem Statment

Given Pointer/Reference to the head of the linked list, the task is to Sort the given linked list using Merge Sort. Note: If the length of linked list is odd, then the extra node should go in the first list while splitting.



Example 1:

Input
N = 5
value[] = {3,5,2,4,1}
Output:
1 2 3 4 5

Method:

  • If,the list contains one or fewer elements, return the same list.
  • Otherwise,Divide the list into halves using the splitting function.
  • Sort both the halves of list
  • Merge the sorted lists

def mergeLists(list1,list2):
    var = None
    if list1 is None:
        return list2
    elif list2 is None:
        return list1
    elif list1.data  <= list2.data:
        var = list1
        var.next =mergeLists(list1.next,list2)
    else:
        var = list2
        var.next = mergeLists(list2.next,list1)
    return  var

class Node:
    def __init__(self, data):
        self.data = data

class LinkedList:                        
    def __init__(self):
        self.head = None
        self.tail = None
                            
                                
    def append(self, new_value):
        new_node = Node(new_value)
        if self.head is None:
            self.head = new_node
            self.tail = new_node    
            return
        self.tail.next = new_node
        self.tail = new_node
